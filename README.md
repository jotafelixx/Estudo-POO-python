# Estudo-POO-python

Projeto simples de RPG em Python executado no terminal, com foco em lógica de programação e Programação Orientada a Objetos (POO).

## Funcionalidades

- Criação de jogador
- Geração de inimigos com nível aleatório
- Sistema de combate por turnos
- Cálculo de dano com base em ataque e defesa
- Recompensa ao derrotar inimigos

## Conceitos aplicados

- Programação Orientada a Objetos (classes e métodos)
- Estruturas de repetição
- Estruturas condicionais
- Modularização de código
- Manipulação de estados (vida, dano, defesa)

## Tecnologias

- Python 3
